This application will show Links the cat from the Clippy.js project with 11 buttons to control it.
9 of the buttons queue and animation to happen and the other 2 stop the current animation, or stop all animations in queue.

To run the program:
    - Extract the submitted .tar file to a folder
    - In that folder, run `flask run --host=0.0.0.0 --port=17025`
    - Navigate to http://cis3210.socs.uoguelph.ca:17025 to access the project
